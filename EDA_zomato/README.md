# 🍽️ Zomato Bangalore Restaurants EDA

Exploratory Data Analysis on Zomato's Bangalore restaurant dataset to uncover dining trends across the city.

## 📌 Objective
To explore restaurant distribution, cuisine popularity, cost patterns, and the effect of online ordering on ratings.

## 📂 Files
```
Zomato/
├── zomato_analysis.ipynb   # Main analysis notebook
└── zomato.csv              # Dataset
```

## 🔍 Analysis Covered
- **Data Cleaning** – Handling nulls, fixing rate column, removing duplicates
- **Univariate Analysis** – Distribution of ratings, votes, cost
- **Bivariate Analysis** – Cost vs Rating, Online order vs Rating
- **Location Analysis** – Top areas with most restaurants
- **Cuisine Analysis** – Most popular cuisines in Bangalore

## 📊 Key Insights
- North Indian and Chinese cuisines dominate Bangalore
- Restaurants with online ordering tend to have higher ratings
- BTM, Koramangala, and Indiranagar have the most restaurants
- Most restaurants fall in the ₹200–₹600 cost range for two

## 🛠️ Tools Used
- Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook
