# 🚢 Titanic Survival Analysis

Exploratory Data Analysis on the Titanic dataset to understand what factors influenced passenger survival.

## 📌 Objective
To analyze survival patterns based on gender, passenger class, age, and fare using visual and statistical exploration.

## 📂 Files
```
Titanic/
├── titanic_analysis.ipynb   # Main analysis notebook
└── titanic.csv              # Dataset
```

## 🔍 Analysis Covered
- **Missing Value Analysis** – Age, Cabin, Embarked columns
- **Univariate Analysis** – Distribution of Age, Fare, Pclass
- **Bivariate Analysis** – Survival vs Gender, Pclass, Age group
- **Data Cleaning** – Dropped quasi-constant and irrelevant features

## 📊 Key Insights
- Females had a significantly higher survival rate than males
- 1st class passengers survived more than 2nd and 3rd class
- Children (age < 10) had better survival chances
- Fare had a positive correlation with survival

## 🛠️ Tools Used
- Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook
